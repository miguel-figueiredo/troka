
# **Troka – Plataforma de Troca para Estudantes Universitários**

## **Objetivo**
Criar uma plataforma para estudantes universitários **trocarem, doarem ou venderem objetos**, incentivando economia, sustentabilidade e interação dentro da comunidade académica.

---

## **Principais Funcionalidades**
- **Cadastro e Login**
  - Autenticação via e-mail institucional ou redes sociais.
  - Perfil com foto, curso, universidade e interesses.

- **Catálogo de Objetos**
  - Publicação de itens com fotos, descrição, estado (novo/usado), categoria (livros, eletrônicos, roupas, material escolar, etc.).
  - Filtros avançados (categoria, localização, preço, condição).

- **Sistema de Troca**
  - Opção para marcar item como “para troca”, “doação” ou “venda”.
  - Propostas de troca entre usuários com chat integrado.

- **Chat Seguro**
  - Mensagens dentro da plataforma para negociação.
  - Notificações push para novas propostas ou mensagens.

- **Avaliações**
  - Sistema de reputação baseado em feedback após cada transação.

- **Localização**
  - Sugestão de pontos de encontro seguros dentro do campus.
  - Geolocalização para encontrar itens próximos.

- **Gamificação**
  - Pontos por cada troca/doação realizada.
  - Rankings e badges para incentivar participação.

---

## **Modelo de Negócio**
- **Freemium**: uso básico gratuito, com recursos premium (ex.: destaque de anúncios).
- **Publicidade direcionada**: marcas voltadas para estudantes.
- **Parcerias com universidades**: integração com eventos e feiras.

---

## **Benefícios**
- Redução de desperdício e incentivo à economia circular.
- Criação de uma comunidade colaborativa.
- Acesso rápido a materiais necessários sem altos custos.
